Microsoft Ergonomic Sculpt Mouse on the Mac

22 November 2015

OVERVIEW

I want to map the mouse keys on my Microsoft Ergonomic Sculpt mouse that came with my new Microsoft Sculpt Erconomic Keyboard. Its a bit of an odd duck of a mouse. It is oddly shaped and a bit uncormforable to use. It is however less uncomfortable than my Apple Magic Mouse.

The main problem I have is that the Windows key does not work with on my Mac. I have found this blog post discussing how to fix it. Per the instructions I installed Karabiner and after a little tinkering was able to map the Windows key to mouse button #5.

From there I used bettertouchtool to map both the thumb buttons to Expose and Mission Control switcher. Below is the private.xml I used in Karabiner.

The mouse is still a tad uncomfortable, but so far so good in terms of mapping the mouse buttons.

PRIVATE.xml

<root>
<deviceproductdef>
<productname>SCULPT_ERGONOMIC</productname>
<productid>0x07a5</productid>
</deviceproductdef>
<item>
<name>Windows Button to Mission Control (Microsoft Sculpt Ergonomic Mouse)</name>
<identifier>com.microsoft.mouse.sculpt_mobile.win_button</identifier>
<device_only>DeviceVendor::MICROSOFT,DeviceProduct::SCULPT_ERGONOMIC</device_only>
    <autogen>
      __KeyToKey__
      KeyCode::OPTION_R,
      KeyCode::VK_MOUSEKEY_BUTTON_BUTTON5
    </autogen>
  </item>
</root>
